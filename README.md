# File-Based Record Management System

A Python-based application that performs CRUD (Create, Read, Update, Delete) operations on structured data stored in JSON format. This project demonstrates the use of file handling and data management without relying on external databases.

## 🚀 Features

* Add new records
* View existing records
* Update specific records
* Delete records
* Data persistence using JSON files

## 🛠️ Tech Stack

* Python
* JSON (for data storage)

## 📂 How It Works

The system stores all records in a JSON file and performs operations directly on it.
Each operation reads the file, modifies the data, and writes the updated content back to ensure persistence.

## ▶️ How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run the Python file:

```bash
python main.py
```

## 📌 Use Case

This project simulates a simple database system using file handling, useful for understanding how data storage and manipulation works at a basic level.

## ⚠️ Limitations

* No GUI (console-based application)
* Not suitable for large-scale data handling
* No authentication or security features
